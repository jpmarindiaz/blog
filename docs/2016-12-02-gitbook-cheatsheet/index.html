Gitbook cheatsheet

December 2, 2016

Create a book

GitBook can setup a boilerplate book:

gitbook init

If you wish to create the book into a new directory, you can do so by running gitbook init ./directory

Preview and serve your book using: gitbook serve Or build the static website using: gitbook build

Generate a PDF file gitbook pdf ./ ./mybook.pdf Generate an ePub file gitbook epub ./ ./mybook.epub Generate a Mobi file gitbook mobi ./ ./mybook.mobi
